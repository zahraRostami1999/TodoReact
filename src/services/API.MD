# How to use

Import Api wherever you want to use it: `import Api from "...../services/api/Api.js"`

Call `Api.Auth.init()` where your application starts to run (before other Api functions).

Run `Api.Auth.isLogedIn()` to see if user is loged in or not (you don't have to use states).

After you type `'Api.'`, Your IDE will list all the usefull functions. If it didn't show anything, press `Ctrl + space`.

I did my best so you don't have to bother with authentication logic. But, it's essential for a frontend developer to understand it all of it.
